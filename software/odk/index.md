The Ontology Development Kit (ODK) helps you set up an ontology project so that it's ready to push to GitHub or other Git repositories.
ODK is Docker-compatible.

For more details, see

 * [2018 Article](https://douroucouli.wordpress.com/2018/08/06/new-version-of-ontology-development-kit-now-with-docker-support/)
